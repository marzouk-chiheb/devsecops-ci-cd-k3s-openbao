\# Grafana Dashboards



The project used Grafana dashboards to visualize:



\## Infrastructure Metrics



\- CPU usage

\- Memory usage

\- Disk usage

\- Network traffic

\- VM availability



\## Application Availability



\- HTTPS endpoint status

\- Response time

\- Application uptime



\## Kubernetes Metrics



\- Pod status

\- Container metrics

\- Deployment health

\- Resource usage



\## Security Logs



\- Security-related logs

\- Application errors

\- OpenBao audit events

\- CI/CD security events
