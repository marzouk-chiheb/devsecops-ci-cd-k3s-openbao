\# Kubernetes Deployment on K3s



This folder contains sanitized Kubernetes manifests used to describe the deployment architecture of the application on a K3s cluster.



The deployment includes:



\- Namespace

\- Backend Deployment and Service

\- Frontend Deployment and Service

\- PostgreSQL Deployment and Service

\- Ingress with HTTPS/TLS support

\- Secret-based environment variables



Sensitive values such as real domains, registry URLs, passwords and tokens were replaced with example values.
