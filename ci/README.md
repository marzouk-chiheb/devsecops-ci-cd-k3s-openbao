\# CI/CD Pipeline



This folder contains a sanitized example of the GitLab CI/CD pipeline used in the project.



The pipeline demonstrates the integration of DevSecOps stages such as:



\- SAST with SonarQube and Semgrep

\- SCA with Snyk and Trivy

\- Secret Detection with Gitleaks

\- Docker image packaging

\- Deployment to K3s

\- DAST with OWASP ZAP

\- Security reporting

\- Secret lifecycle monitoring



Sensitive information such as tokens, credentials, URLs and internal configuration values were removed for security reasons.
