\# Monitoring and Observability



The project includes a monitoring stack to supervise the infrastructure and the deployed application.



\## Tools



\- Prometheus for metrics collection.

\- Grafana for dashboards.

\- Loki for log aggregation.

\- Alertmanager for alert notifications.

\- Node Exporter for VM metrics.

\- Blackbox Exporter for availability checks.

\- cAdvisor for container metrics.



\## Alerts



The monitoring system includes alerts for:



\- VM down

\- High CPU usage

\- High memory usage

\- Low disk space

\- Application unavailable



\## Objective



The objective is to provide visibility into the health, availability and performance of the environment.
